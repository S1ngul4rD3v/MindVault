# MindVault

[![Tests](https://github.com/S1ngul4rD3v/MindVault/actions/workflows/tests.yml/badge.svg)](https://github.com/S1ngul4rD3v/MindVault/actions/workflows/tests.yml)

Proyecto Node.js (ESM) con Express.

## Scripts

- `npm run dev` — desarrollo con nodemon
- `npm start` — ejecutar aplicación
- `npm test` — tests con Jest
- `npm run test:watch` — tests en modo watch
- `npm run test:coverage` — tests con cobertura

## Configuración

Copiar `.env.sample` a `.env` y ajustar variables.

## API

- `GET /health` — health check
- `GET /api-docs` — documentación Swagger
